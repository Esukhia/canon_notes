# ལས་རིམ་སྙིང་བསྡུས་ཨིན་བོད་ཤན་སྦྱར་མ།
སྤྱིར་ལས་མི་དུ་མས་བསྒྲུབ་དགོས་པའི་བྱ་བ་གང་ཡིན་རུང་། དེ་ལ་སྒྲུབ་འབྲས་ཐོན་མི་ཐོན་ནི་འཆར་གཞི་ཡོད་མེད་དང་། ལས་མི་རེ་རེ་འཆར་གཞི་དེ་ལག་སྟར་བྱེད་ཐུབ་མིན་ལ་ཐུག་ཡོད། དེ་ཡང་ལས་འཆར་ལ་རྒྱས་བསྡུས་དང་རག་ཞིབ་གྱི་དབྱེ་བ་སྣ་ཚོགས་མཆིས་ཀྱང་། དེ་དག་སྙིང་པོར་སྒྲིལ་ན་བཞི་སྟེ། 1. ཐོག་མའི་རྒྱུ་ཆ་ནི་གང་། 2. དེར་བྱེད་པའི་བྱ་བ་ནི་ཅི་དང་བྱེད་མཁན་ནི་སུ། 3. འབྲས་བུའི་ཁྱད་ཆོས་ནི་ཅི། 4. ཕྲད་པའི་དཀའ་ངལ་ནི་དུ་དང་སེལ་ཐབས་ནི་གང་། ཞེས་དང་བཞི་ཡིན། འདིར་པཎ་གྲུབ་བཅུ་བདུན་གྱི་གསུངས་འབུམ་བསྡུ་སྒྲིག་ཞུས་པའི་ལས་རིམ་སོ་སོར་སྦྱར་ན།
